# Privacy Policy - Firefox Workspaces Extension

**Last Updated:** February 2, 2026

## Summary

Firefox Workspaces does **not** collect, transmit, or share any of your data with external servers. All data stays within your Firefox browser and syncs only through Mozilla's Firefox Sync service.

## What Data Is Stored

The extension stores the following data locally in your browser:

| Data | Purpose | Storage Location |
|------|---------|------------------|
| Workspace names | Identify your workspaces | Firefox Sync (storage.sync) |
| Tab URLs | Restore tabs when opening a workspace | Firefox Sync (storage.sync) |
| Tab titles (optional) | Display in workspace list | Firefox Sync (storage.sync) |
| Pinned tab state | Restore pinned tabs correctly | Firefox Sync (storage.sync) |
| User preferences | Remember your settings | Local storage (storage.local) |

## How Data Is Synced

- Data syncs **only** through Mozilla's Firefox Sync service
- Sync requires you to be signed into a Firefox Account
- Mozilla's [Firefox Sync Privacy Notice](https://www.mozilla.org/privacy/firefox/#sync) applies to synced data
- No data is sent to any other servers

## Data We Do NOT Collect

- ❌ Page content or browsing history
- ❌ Personal information (name, email, etc.)
- ❌ Analytics or usage statistics
- ❌ Crash reports
- ❌ Any data sent to third parties

## Permissions Explained

| Permission | Why It's Needed |
|------------|-----------------|
| `tabs` | Read tab URLs and titles to save in workspaces |
| `storage` | Save workspace data locally and via Firefox Sync |
| `menus` | Add "Move to Workspace" context menu on tabs |

## Data Deletion

To delete all extension data:
1. Open extension Settings
2. Click "Clear All Workspace Data"

Or uninstall the extension, which removes all local data.

## Private Browsing

This extension does **not** access private browsing windows. Tabs from private windows are never captured or stored.

## Contact

This extension is for personal use. For questions or concerns, please open an issue on the project repository.
